To check the wav file in the testing_team folder, just run the below code in commandline. The prediction uses the pre-trained model dog-bark.pickle for the prediction.
```bash
python DogBarkPredicting.py
```

Requirements:
- Python 3.6
- Tensorflow 1.14.0
- IPython
- librosa
- libsndfile1
- keras==2.3.1
